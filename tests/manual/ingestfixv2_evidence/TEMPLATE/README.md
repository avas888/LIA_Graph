# Ingestfix-v2 Evidence Bundle Template

Copy this directory to `../<run-ts>/` (e.g. `../2026-04-21T1500Z/`) and
fill in the files listed in `tests/manual/ingestfixv2_e2e_runbook.md` §8.

Each run of the E2E runbook produces one bundle. Keep them in git so the
stakeholder sign-off trail stays reproducible.
